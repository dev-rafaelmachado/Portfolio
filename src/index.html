Block Note

Uma aplicação em Electron que guarda suas notas em um sistema de blocos com conexão ao Firebase para manter as anotações guardadas. Utiliza tecnologias como: Node.js, JS, HTML, CSS, Firebase que faz parte do sistema do cloud da google.

RPG Controller

Uma aplicação em Electron que ajuda a gerir os atributos do seu personagem no RPG de mesa. Utiliza tecnologias como Node.js, JS, HTML, CSS e um sistema de arquivos JSON para persistência de dados.

PokedexJS
